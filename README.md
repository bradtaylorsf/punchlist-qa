# Punchlist QA

Standalone QA testing dashboard and support widget toolkit for any project.

## What It Does

- **Support Widget** — Drop-in floating FAB, inline component, or menu item that lets users submit tickets to GitHub Issues, Jira, or Linear
- **QA Dashboard** — Standalone app for running structured QA test rounds with history tracking, tester attribution, and issue auto-filing
- **AI-Powered Setup** — `npx punchlist-qa init` scaffolds config and installs Claude Code or Codex skills that auto-generate test cases from your codebase

## Quick Start

```bash
npx punchlist-qa init
```
